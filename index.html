Let's first discuss what my project is even going to be about. My project involves programming and training a neutral network AI that will be installed on a raspberry Pi. This will be used to control the voltage output of a hydropower system and this hydropower system will then power on a desktop computer. Some things to keep in mind is that the amount of water is finite and may not last unless if an external water source is connected, such as a water hose.

I have also expanded the scope of this project - not only will the project use a neural network AI but the machine learning will be advanced. Furthermore, other features will be included, such as dynamic load balancing. This project will also be documented.
While this project is incredibly innovative, some of the risks include the following:
- Electrocution
- Fire hazards
- Potential lead poisoning (if the solder used is leaded)

Here are some of the safety measures:
- Wearing gloves, goggles, a mask
- Grounding yourself (wearing Grounding straps, etc.)
- Not using leaded solder
